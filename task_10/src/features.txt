Список реализованных фич:
+ 1. Добавление договора с заданным номером (строка) и датой заключения (строка формата "YYYYMMDD", где Y - год, M - месяц, D - день)
+ 2. Возможность регистрировать платежные документы типа "Платежное поручение" и "Банковский ордер"
+ 3. Добавление платежного документа с указанной суммой, номером, типом документа, номером договора и датой платежа. Сумма задается целым положительным числом копеек; номер платежного документа - положительное целое число; дата платежа - строка формата "YYYYMMDD"; тип документа задается перечислением
+ 4. Поиск всех платежей по договору с заданным номером
+ 5. Вычисление суммы всех платежей по договору с заданным номером
+ 6. Удаление платежей с заданным номером, номером договора и датой (только по номеру договора)
+ 7. Получение списка всех договоров с их суммарными суммами платежей
+ 8. Получение списка всех платежей
